To setup the compiler in wsl and to compile’ Feature’ you need to execute in the bash shell : 

sudo apt-get install make
sudo apt-get install libz-dev
sudo apt-get instal g++
sudo apt-get update
sudo apt-get instal g++
unzip feature file after download and copy to ubuntu with ‘ > explorer.exe . ’ executed in bash shell.
Then :
>  tar -xvzf feature-3.1.0-src.tar.gz
>  cd feature-3.1.0
> make
> sudo make install
